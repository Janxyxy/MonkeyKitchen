# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly by emailing **[your-email@example.com]**.  
Do **not** open an issue on GitHub for security-related reports. This helps prevent public exposure before a fix is available.

## Supported Versions

This project is a learning/demo project and does **not** have production releases. Security fixes will be applied only when feasible.

## Security Best Practices

- This project uses Unity and the free assets provided by the [Kitchen Chaos Course](https://unitycodemonkey.com/kitchenchaoscourse.php).  
- Ensure you are using a recent version of Unity to avoid known vulnerabilities.  
- Do **not** use this project in production with untrusted network connections without proper security audits.  

## Disclaimer

This project is intended for educational purposes. The author is **not responsible** for any misuse or security issues arising from deploying this project in production environments.
